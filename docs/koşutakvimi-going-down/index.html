Koşutakvimi Going Down

July 21, 2015

This is to note that simple calendar I created to list all running events in and around Turkey is no more being maintained. kosutakvimi.org was one of the first neat lists to do its task. It was introduced in the only Turkish running podcast. Here are several good alternatives if you want to give it a try:

koscora.com/yaris-takvimi

yaristakvimi.com

kosukolik.com/etkinlik-takvimi/

Runnery for Android

Runnery for iOS
