Pull Request Kulübü 2019 Raporu

CPAN Meydan Okuması CPAN geliştiricileri ile katkı sağlamak isteyen katılımcıları eşleştiriyordu. Katılımcılar aylık ödev almak için kayıt oluyordu. Hedef ise en azından 1 pull request göndermekti. Bu çok eğlenceliydi, ama 2018 sonunda sona erdi.

Ben bu meydan okumanın devam etmesini istediğim için Pull Request Kulübü’nü oluşturdum. Başladığımızdan beri tam bir yıl geçtiği için yıllık bir rapor yazıp sayıları paylaşmak istedim.

Yeni özellikler

  • Pull Request Kulübü temel özelliklerini CPAN Meydan Okuması’ndan alıyor ancak ben biraz daha self servis bir site yaratmak istedim. Kayıt olmak veya bir ödevi atlamak gibi işlemler artık birkaç tıklama ile yapılabiliyor.
  • Kullanıcılar siteye GitHub hesaplarıyla kayıt oluyorlar. Böylece kendi depolarını çok kolay bir şekilde ödev havuzuna ekleyebiliyorlar.
  • Kullanıcılar sitede hem aldıkları hem de verdikleri ödev geçmişlerini görebiliyorlar.

Pull Request Kulübü, Benim Ödevim sayfası.

Sayılar

  • Havuzda 100’den fazla kullanıcımız ve 200’den fazla depomuz oldu.
  • 70 kullanıcımız en az 1 ödev aldı.
  • 16 kullanıcımız en az 1 ödevini tamamladı.
  • Sadece 1 kullanıcımız 12 ödevin hepsini tamamladı.
  • Ekim ayı 8 ödev ile en çok ödevin tamamlandığı ay oldu. Ekim’i 7 ödev ile Ocak ve Mart izledi.
  • 21 kullanıcımızın en az 1 deposu ödev olarak verildi.
  • 14 kullanıcımız en az 1 pull request aldı.
  • Bu yıl 170’ten fazla ödev dağıttık.
  • Bu ödevlerin 60 tanesi tamamlandı. Yani kullanıcılarımız 60 pull request gönderdi.

Bize katılın!

Kulağınıza ilginç geldi mi? Yeni yılda açık kaynak dünyasına siz de katkı sağlamak ister misiniz? Pull Request Kulübü’ne hemen bugün katılabilirsiniz.

Sadece depolarınıza gelecek pull requestler ile ilgileniyorsanız da kaydolabilirsiniz. Kayıt olduktan sonra ödev almak istemediğinizi belirtseniz bile depolarınızı ödev havuzunda tutabilirsiniz.

Yeni yıldan önce kayıt olursanız (ve ödev almak istediğinizi belirtirseniz), ilk ödevinizi 1 Ocak’ta alabilirsiniz.
