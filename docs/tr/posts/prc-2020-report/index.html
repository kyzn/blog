Pull Request Kulübü 2020 Raporu

Pull Request Kulübü, açık kaynak geliştiricileri ile katılımcılarını eşleştiren bir web uygulamasıdır. Bu eşleştirme aylık ödevler ile gerçekleşir. Geliştiricilerin kod depoları katılımcılara ödev olarak verilir. Katılımcıdan pull request hazırlaması istenir.

Pull Request Kulübü CPAN Meydan Okuması’nın devamı olarak geliştirildi. 2020 yılının sona ermesiyle Pull Request Kulübü 2. yılını tamamladı. Burada 2. yıllık raporu okuyabilirsiniz. Geçen yılın raporuna buradan erişebilirsiniz.

Yeni özellikler

  • Yeni üyelere kayıt olur olmaz bir hoş geldiniz ödevi veriliyor. Bu ödev 36 kez verildi ve 9 kez tamamlandı. Hoş geldiniz ödevini tamamlayan 4 kullanıcı başka ödevleri de başarıyla tamamladı.
  • Kullanıcılar uygulamadan hangi emailleri alacaklarını seçebiliyorlar.
  • Ayarlar sayfasına sekmeler eklendi. Daha önceden bütün ayarlar tek bir sayfada seçiliyordu.
  • Çatal yapılarak oluşturulmuş depolar da artık ödev havuzuna eklenebiliyor.

İyi haber: Pull Request Kulübü üyeleri 133+ pull request gönderip aldılar! 🎉 Ödev tamamlama oranı da biraz yükseldi.

Ödev alan katılımcılar (123) ile PR göndermiş katılımcılar (29) arasındaki farka dikkatinizi çekerim. Kayıt olup, ödev aldıktan sonra siteyi terkeden 94 kişi var. Bu kişileri siteye geri getirmenin yollarını arıyorum.

En iyi üç kullanıcı

Bazı kullanıcılarımız açık kaynak dünyasına olan sevgileri ile rekorlar kırıyorlar. Burada kısaca onlardan bahsetmek isterim.

  • Mohammad S Anwar - Mohammad 24 defa ödev alıp 24’ünü de tamamlayan tek kullanıcı! Müthiş. Önümüzdeki yıllık raporda bu sayının 36 olacağından eminim.
  • Håkon Hægland - Håkon Pull Request Kulübü’ne Mart 2019’un sonlarında katıldı. Kayıt olduktan sonra 21 ödev alıp hepsini tamamladı! Harika.
  • E. Choroba - Choroba tam 18 ödev tamamladı, harika bir sayı! Ödev tamamlamaya ek olarak Choroba’nın depoları da 15 kez diğer kullanıcılara ödev olarak verildi. Kendisi hem harika bir katılımcı hem de harika bir geliştirici.

Bütün Pull Request Kulübü üyelerine açık kaynağa katkılarından dolayı teşekkür ederim.
